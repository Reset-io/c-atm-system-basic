# 💳 ATM System in C (Beginner Project)

A simple ATM simulation built using C. This project demonstrates core programming concepts like loops, conditionals, and user input handling.

---

## 🚀 Features

- Check account balance
- Deposit money
- Withdraw money
- Menu-driven interface
- Continuous loop until user exits

---

## 🧠 Concepts Learned

- `while` loops (infinite loop using `while(1)`)
- `if-else` decision making
- User input using `scanf`
- Variable handling and state management
- Basic program flow control
- Difference between `break` and `return`
- Real-world logic simulation (ATM behavior)

---

## ⚠️ Known Issues / Future Improvements

- Program may crash if user enters non-integer input (e.g., alphabets)
- No validation for negative inputs
- No PIN/authentication system
- No transaction history

---

## 🛠️ Tech Used

- C Programming Language
- Standard Input/Output Library (`stdio.h`)

---

## 🤝 Learning Approach

This project was primarily built independently by understanding logic step-by-step.

- AI was used only for:
  - Direction and guidance
  - Debugging logic mistakes
  - Understanding concepts (not direct code copying)

Most of the logic, structure, and problem-solving was done manually.
